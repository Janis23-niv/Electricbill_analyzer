Electric Bill Analyzer 

A Java Swing and MySQL-based desktop application for managing electricity billing operations efficiently. The project includes secure admin/customer login, customer management, bill generation, electricity usage tracking, and database connectivity using JDBC.

🔹 Features

•	Admin and Customer Login Authentication 

•	Add, Update, Delete, and Search Bills 

•	Customer Management System 

•	Electricity Usage Tracking 

•	Automatic Bill Amount Calculation 

•	MySQL Database Integration using JDBC 

•	Interactive GUI using Java Swing 

🔹 Technologies Used

•	Java 

•	Java Swing 

•	JDBC 

•	MySQL 

•	Eclipse IDE 

🔹 Database

MySQL database used: electricbilldb
